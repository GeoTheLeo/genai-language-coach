# 🌍 GenAI Language Coach

A multimodal AI-powered language learning system combining LLMs, speech recognition, and pronunciation scoring.

---

## 🚀 Features

- ✍️ Text-based language practice
- 🎤 Speech-to-text interaction
- 🔊 IPA pronunciation guidance
- 🎯 Pronunciation scoring (Wav2Vec2)
- 🧠 LLM-powered structured outputs
- 📦 Vector database (Chroma)
- 📊 LangSmith tracing support

---

## 🧠 Tech Stack

- FastAPI
- Streamlit
- LangChain
- OpenAI API
- HuggingFace (Wav2Vec2)
- ChromaDB
